# Grenadier

Login gateway for phoenix umbrella app.

## Custom Mix Tasks

* Create user `mix grenadier.create_user`

* Update user's password `mix grenadier.update_user`

* Delete user `mix grenadier.delete_user`

## Super Search

* Set `UMBRELLA_SUPERSEARCH_IP_REGEX` and `UMBRELLA_SUPERSEARCH_ORG_REGEX` environment variables to regex patterns for Super Search alert classes based on IP
